# CSS选择器
CSS 选择器是一种 通过 HTML 元素的标签名、class、id、属性等特征 来定位页面元素的方式。
它比 XPath 更简洁、易读，且被现代浏览器和自动化工具广泛优化支持

#

| 选择器 | 含义 | 示例 | 匹配元素 |
|--------|------|------|--------|
| `*` | 所有元素 | `*` | 页面上每一个标签 |
| `tag` | 标签名 | `div`, `p`, `a` | 所有 `<div>`、`<p>` 等 |
| `.class` | class 属性 | `.title` | `<h1 class="title">` |
| `#id` | id 属性 | `#main` | `<div id="main">` |
| `[attr]` | 有某属性 | `[href]` | 所有带 `href` 的 `<a>` |
| `[attr=value]` | 属性等于值 | `[type="submit"]` | `<input type="submit">` |

| 选择器 | 含义 | 示例 | 说明 |
|--------|------|------|------|
| `A B` | 后代（任意层级） | `.content p` | `.content` 内所有 `<p>` |
| `A > B` | 直接子元素 | `ul > li` | 只匹配 `ul` 的直接 `li` 子项 |
| `A + B` | 紧邻兄弟 | `h1 + p` | 紧跟在 `<h1>` 后的 `<p>` |
| `A ~ B` | 通用兄弟 | `h1 ~ p` | `<h1>` 后所有同级 `<p>` |
